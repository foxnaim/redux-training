## About

A training project focused on learning Redux state management. Covers core concepts like store, actions, reducers, middleware, and best practices.

## Topics Covered

- 🏪 **Store** — Single source of truth
- 📤 **Actions** — Describing state changes
- 🔄 **Reducers** — Pure functions for state updates
- 🔌 **Middleware** — Async operations with thunks
- 🪝 **React-Redux** — Hooks (useSelector, useDispatch)

## Getting Started

```bash
git clone https://github.com/foxnaim/redux-training.git
cd redux-training
yarn install
yarn dev
```
